# Support

## Questions

Use GitHub issues or the documented contact channel for scoped questions about
setup, verification, or contribution workflow.

## Bugs

Open a bug issue with:

- expected behavior,
- actual behavior,
- reproduction steps,
- environment,
- logs or screenshots with secrets removed.

## Security

For vulnerabilities, follow `SECURITY.md`.

## Scope

This project is maintained as a portfolio/collaboration artifact. Support is
best-effort unless stated otherwise.
